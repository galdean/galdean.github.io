One-Button Accessibility Chrome Extension

A Google Chrome extension designed to provide accessibility for individuals with motor issues and physical disabilities. This extension enables users to navigate websites using only one button, such as the spacebar, simplifying the browsing experience.

Magnifier Chrome Extension

This project involves building a web-based magnifier using a Chrome extension to assist visually-impaired users.

  • Page zooms out using -/_ key and zooms in using +/= key.
  • Page graphically zooms out using shift -/_ keys and in using shift +/= keys.
  • Horizontal scrolling when the mouse is moved to the edges of too zoomed in screen
  • Page elements are displayed using the spacebar key and pressing space again dismisses them.
